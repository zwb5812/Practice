# 采购文件智能生成网页原型

这是一个静态前端原型，用于演示“需求输入 -> 语义解析 -> 模板匹配 -> 文件生成 -> 风险预审 -> 文件输出”的采购文件智能生成流程。

## 文件结构

```text
Project_1
├── index.html   页面结构
├── styles.css   页面样式
├── app.js       交互逻辑、模拟解析、模拟生成、风险预审、导出
└── README.md    项目说明
```

## 功能

- 支持录入采购需求标题、需求内容、预算、交付周期、供应商要求。
- 模拟大模型将非结构化需求解析为结构化招标参数。
- 模拟匹配招标公告、招标文件主体、合同条款、评审标准模板。
- 自动生成采购文件预览。
- 自动进行合规风险预审。
- 支持查看生成记录、重新生成、局部调整。
- 支持导出 Word 格式文件。

## 打开方式

直接用浏览器打开 `index.html` 即可。

后续如需接入 Spring Boot 后端，可以将 `app.js` 中的模拟函数替换为接口请求。
